# BedrockPro
## これなに
Minecraft BedrockEditionのリソースパックとかを簡単に生成できるようになる(予定)
そんな感じのツールになればいいなぁ...
## 現在実装されている機能
- ResourcePackGen
  - リソースパックを生成
    - MCPEのapkを解凍して生成
    - UUIDの自動生成
    - カスタムアイコン
    - デフォルトリソースをキャッシュ
- BackupAPK
  - MCPEのapkをバックアップする
